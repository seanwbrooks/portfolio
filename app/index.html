My Beginning | Portfolio | Bios

My journey as a web developer began in 2017.

In Intro to Computer Science in 2009, I was introduced to programming in C++.

Here’s how I got started…

Over the course of my professional life, I began to see a need for computer programs to automate the tasks and projects I was given. So I started to learn Visual Basic for Applications, a programming language for Excel, to fill in this need.

It was a lot of fun building programs from scratch.

When I moved to Boston in the fall of 2016, I was presented with a great opportunity to start a career in web development. I enrolled into Launch Academy, one of Boston's oldest coding bootcamp, where I became fluent in Javascript and Ruby.
